# ❤️ Heart Disease Prediction App

A Machine Learning-based web app built using **Streamlit** that predicts the risk of heart disease based on user inputs.

---

## 🚀 Features
- Predicts **heart disease risk (High / Low)**
- Simple and interactive UI
- Built using **KNN (K-Nearest Neighbors)**
- Real-time prediction
- Data preprocessing with scaler

---

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Joblib
